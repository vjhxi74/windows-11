## 买了之后怎么用？

数码荔枝是自动发货，付款后激活码会发到网页、短信和邮箱，不用等人工审核。

激活流程也不复杂：

1. 点开始菜单，搜索「设置」并打开
2. 进入「系统 → 激活」
3. 点「激活 / 更改」，输入激活码
4. 等几分钟，激活完成

激活后建议登录微软账号绑定数字权利——这样换电脑的时候，授权可以迁移，不用重复购买。

## Windows 11 有什么值得说的新东西？

很多人停留在 Win10 的认知里，觉得 11 也没什么大变化。实际上体验差还是有的：

**界面变了**。开始菜单居中，任务栏更干净，整体视觉现代了很多，第一眼真的挺好看的。

**多窗口布局终于内置了**。拖动窗口到顶部就能快速选择分屏方式，以前要装第三方软件才能做到的事现在系统自带了。

**虚拟桌面更好用**。工作一个桌面，学习一个桌面，娱乐一个桌面，切换干净，互不干扰。

**游戏体验有提升**。Auto HDR、DirectX 12、DirectStorage（直通存储）几个技术组合，加载速度和画面都有提升，尤其对新游戏友好。

**安全性更强**。TPM 2.0 加密、安全启动、Windows Defender，几层叠加，比 Win10 的防护体系完整很多。

## 装机必看：这几个问题最常被问到

**Q：家庭版能升级到专业版吗？**
可以。在「设置 → 系统 → 激活 → 更改产品密钥」里输入专业版密钥，系统会自动升级。也就是说，先买家庭版用着，以后有需要可以补差价升级专业版。

**Q：买了会不会掉激活？**
正版数字权利绑定微软账号，不会无缘无故掉。极端情况比如主板坏了换新板，重新登录微软账号一般也能自动激活。

是的，数码荔枝出售的 Windows 11 激活码仅限中国地区使用，这一点在商品说明里有明确标注，购买前需要留意。

不行，一码对应一台设备，限购也是每单最多 2 件。有多台需求需要多次购买。
